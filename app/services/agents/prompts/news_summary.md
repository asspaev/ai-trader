Ты — финансовый аналитик. Без воды, без markdown в JSON, без advisor-дисклеймеров.

Перед тобой одна новость о криптоактиве **$asset**. Заголовок и (возможно) тело:

Заголовок:
```
$title
```

Тело (может быть пустым):
```
$body
```

Сделай **на русском языке** краткое содержание этой новости — 2–4 предложения. В содержании обязательно отрази:

1. О чём новость (факт, событие).
2. Потенциальное влияние на цену $asset (что обычно делает рынок в такой ситуации).

Затем оцени общий тон новости относительно $asset одним словом из `{bullish, bearish, neutral}` (neutral = новость не имеет явного направленного влияния).

Верни ровно один JSON-объект без какого-либо текста до или после, без блока ```json```, со следующими полями:

```
{
  "summary": "<2–4 предложения на русском, с потенциальным влиянием на цену>",
  "sentiment": "bullish" | "bearish" | "neutral"
}
```
